# Lean 4 Formalization

This directory contains the Lean 4 formalization for *Characterizing Arithmetic Through Dirichlet Series with Boolean Coefficients*.

See the [repository README](../README.md) for the full paper--formalization correspondence, axiom documentation, and build instructions.

## Quick Build

```bash
lake build
```

Requires Lean 4.24.0 (see `lean-toolchain`). Mathlib is fetched automatically.
